# 📊 Deloitte Australia Data Analytics Virtual Experience
## 📖 Overview

This repository contains my solutions for the **Deloitte Australia Data Analytics Virtual Experience Program** hosted on **Forage**.

The project involved solving real-world business problems using **Tableau** and **Microsoft Excel** by analyzing business data and presenting actionable insights through dashboards and data classification.

## 🎯 Client Requirements

# 📌 Task 1 – Tableau Dashboard

The client wanted to:

- Analyze telemetry data collected every 10 minutes from four Daikibo factories.
- Identify the factory with the highest machine downtime.
- Determine which device types contributed the most downtime.
- Build an interactive Tableau dashboard with KPIs and visualizations.

# 📌 Task 2 – Forensic Technology

The client wanted to:

- Analyze gender pay equality data using Microsoft Excel.
- Develop a classification system based on the provided business rules.
- Transform raw data into meaningful business insights.
